# Space-Debris-Collision-Avoidance-System

A conceptual solution developed for a DSA Ideathon to address the growing problem of satellite collisions caused by space debris. The project proposes an intelligent decision-support system that predicts potential collision risks and generates optimal avoidance paths using heuristic search algorithms such as **A\***.

---

## Problem Statement

The increasing amount of space debris has made satellite collisions a major challenge for space missions. Even tiny debris traveling at orbital speeds can cause catastrophic damage.

This project focuses on predicting possible collision paths and recommending safe, fuel-efficient avoidance maneuvers to improve satellite safety and mission reliability.

---

## Objectives

- Predict potential satellite–debris collisions.
- Generate safe avoidance paths.
- Minimize fuel consumption during maneuvers.
- Reduce mission disruption.
- Provide an automated decision-support system for satellite operators.

---

## Proposed Solution

The system follows these steps:

1. Model satellites and debris using positional data.
2. Detect possible collision risks through trajectory analysis.
3. Apply the **A\*** heuristic search algorithm to determine the optimal avoidance path.
4. Recommend safe maneuvers while minimizing fuel usage.
5. Produce outputs that can be integrated into mission-control workflows.
